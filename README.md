# Sistema de Gestão de Igreja

Aplicação web para gestão administrativa de igreja/comunidade religiosa.

## Tecnologias
- Next.js (React)
- Tailwind CSS
- Firebase (Firestore)
- JWT Auth
- Hospedagem: Vercel

## Estrutura de Pastas
- src/components: Componentes compartilhados
- src/pages: Rotas e páginas
- src/styles: Estilos globais
- src/contexts: Contextos globais (auth, tema, etc.)
- src/hooks: Hooks customizados
- src/services: Integração com Firebase
- src/assets: Ícones, imagens
- src/modules: Domínios do sistema (dashboard, financeiro, eventos, estoque, usuários)

## Funcionalidades
- Dashboard financeiro e de eventos
- Controle de caixa e mensalidades
- Gestão de estoque (velas, materiais)
- Gestão de eventos/cultos
- Controle de usuários e permissões
- Layout moderno, responsivo, menu lateral fixo

## Como rodar
1. Instale as dependências
2. Configure o Firebase
3. Rode o projeto com `npm run dev` ou `yarn dev`
